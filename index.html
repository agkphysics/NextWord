NextWord The predictive text game

Help/About

Help

NextWord is a game in which you have to guess what word is most likey to come next in the phrase given.
Your score for each word input will be a log-probability, that is, the logarithm of the probability that your word comes next. This is a negative number; the lower the probability, the more negative the number.
Your score is the logsum of all log-probabilities you have gotten so far, so the greater the number (less negative), the better your score.


About

This web app uses the Microsoft WebLM API and is hosted on the Azure platform.
